# Icon 图标

Vue.js 的 语义化的矢量图形。

# 如何使用

使用 <w-icon /> 标签声明组件，指定图标对应的 type 属性，示例代码如下:

<w-icon type="forward" />

# 水滴里面的 icon

示例类型代码
向前<w-icon type="forward" />
向后<w-icon type="back" />
加载中1<w-icon type="loading1" />
加载中2<w-icon type="loading2" />
加载中3<w-icon type="loading3" />
加载中4<w-icon type="loading4" />
心<w-icon type="heart" />
星星<w-icon type="star" />

# 本地部署

图标默认托管在 iconfont.cn ，默认公网可访问。如需本地部署，可参考 示例 。

# API

由于图标字体本质上还是文字，可以使用 style 和 class 设置图标的大小和颜色。

参数说明类型是否必填默认值
type图标类型。如果为空则不显示String是无
spin是否有旋转动画Boolean否无
prefix样式的前缀String否无
